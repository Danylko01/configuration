这是一个关于vim配置文件(.vimrc)和tmux配置文件(.tmux.conf)的仓库
在insert模式下使用jj将normal模式
将空格将设置为leader

使用leader + w直接写入

插件 nerdtree
leader  + o显示文件目录
leader + p 从文件回到目录栏中

插件ctrlp
使用 ctrl + p快速搜索文件

插件vim-easymotion
使用ss + 对应字母 在当前窗口快速移动光标

插件tarbar
使用 leader + t  快速浏览代码，显示函数变量等

插件vim-commentary
使用gcc快速注释/取消注释代码

插件far.vim
进行替换
